# gpxtogjsonmap
Startpunkt: GPX-Datei

Ziel: Konvertierung in GeoJSON (+ Website bzw. Kartendarstellung mit Zusatzinformationen)

Used: Vue, Vue-Bootstrap, Vue-Leaflet, Vue-Router



## Project setup

```
yarn install
```
